League Profile

A personalized profile and match history for League of Legends

This League Profile application was written in Java, using Riot's API.

The program features a profile search function, which then pulls information for the associated account.

Their accounts current Ranked Data, player level, and top 3 champion mastery scores are pulled, as well as their most recent 20 games for their match history.

Right clicking on a match, you can select more for more details, which brings up a second window featuring more statistics such as game length, K/D/A, cs scores, and the player names and champions selected by all 10 players in the match.

Here's a small sample of the code to send an API request using Riot's API:

                            
    public static Summoner getPlayerData(String regionCode, String summoner) {
        String version = "v4";

        String urlSummoner = "https://" + regionCode + ".api.riotgames.com/lol/summoner/" + version +
                "/summoners/by-name/" + summoner + "?api_key=" + LeagueProfileAPI.RIOT_API_KEY;
        Summoner player = Unirest.get(urlSummoner)
                .asObject(Summoner.class)
                .ifFailure(Error.class, r -> {
                    Error e = r.getBody();
                })
                .getBody();
        return player;
    }
